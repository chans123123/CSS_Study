전체 선택자 : * {CSS } : 모든 요소를 하나의 그룹으로 묶음
태그 선택자 : 태그명 {CSS} : 태그로 작성한 요소를 지정
아이디 선택자 : id 속성값 {CSS} : id 속성값인 요소를 지정
클래스 선택자 : .class속성값 {CSS} : class 속성값인 요소를 지정
그룹 선택자 : 선택자, 선택자, 선택자 {CSS} 여러 선택자를 하나의 그룹으로 묶음
자식 선택자 : 부모 선택자 > 자식 선택자 {CSS} 부모 요소의 하위요소에 있는 자식요소를 지정
하위 선택자 : 선택자 선택자 {CSS} 선택자의 범위를 부모 요소의 하위 요소로 한정
인접 형제 선택자 : 이전 선택자 + 대상 선택자 {CSS} 이전 선택자와 인접한 형제 대상선택자를 지정
일반 선택자 : 이전 선택자 ~ 대상 선택자 {CSS} 이전 대상자와 형제 관계에 있는 모든 요소들을 선택자로 지정
