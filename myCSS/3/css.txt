::before : 선택자 기준 앞 공간 선택
::after : 선택자 기준 뒤 공간 선택
content : ::before, ::after선택자에서만 사용되는 속성, 새로운 요소를 만들거나 추가할 때 사용
:link : 한번도 방문하지 않은 사이트를 선택
:visited : 방문한적이 있는 링크를 선택
:hover 요소에 마우스 커서를 올리면 태그가 선택자로 지정
:active : 클릭하고 있는 동안 선택자로 지정
:focus : 입력요소에 커서가 활성화 되면 선택자로 지정
:checked : 체크박스에 커서가 활성화 되면 선택자로 지정
:disabled : 비활서오하 되면 선택자로 지정
